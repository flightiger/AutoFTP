##AutoFTP是什么?
在任意位置建立文件夹，在这个文件夹放入任何文件，将实时同步传输到远程FTP服务器。

##AutoFTP有哪些功能？

* ini文件设置全部参数
    *  直接修改服务器ip、账号密码
    *  直接修改本地监听文件夹路径
